# Sunny
A Sony Playstation emulator, The motivation behind this project was to learn about emulation and hardware, and to improve my skills at low-level programming. Everything in this project was made from scratch, without any third-party library sans the OS-specific platform code.

## Build
Sunny is currently only supported on Windows, The build system uses my header-only build library which is simple to use, A batch file is also provided for convenience.

Note that a valid BIOS file is required to run the emulator.

## Run
Press the 'M' key to bring up a debug menu.

Images can be loaded directly using the menu, or specified on boot using a sunny.cfg config file
